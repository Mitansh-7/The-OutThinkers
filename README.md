# Pixel Flow — The OutThinkers

A premium fashion e-commerce storefront built with **vanilla HTML, CSS & JavaScript**.

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero banner + featured collection |
| Women | `women.html` | Women's collection with sidebar filters |
| Men | `men.html` | Men's collection with sidebar filters |
| New Arrivals | `new-arrivals.html` | Latest drops |
| Sale | `sale.html` | Discounted items with sale badges |
| Search | `search.html` | Dynamic live search with filters |
| Cart | `cart.html` | Shopping bag with quantity controls |
| Account | `account.html` | User profile page |
| About | `about.html` | Brand story |

## Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties, grid, flexbox, animations
- **JavaScript (ES6)** — vanilla, no frameworks
- **Font Awesome 6** — icons
- **Google Fonts** — Inter + Playfair Display

## Features

- 🛒 **Dynamic Cart** — add/remove items, quantity +/-, live price recalculation
- 🔍 **Live Search** — real-time filtering with category & price range filters
- 👕 **Size Selection** — premium size picker (XS–XXL) on add-to-cart
- 💰 **INR Pricing** — all prices in Indian Rupees (₹)
- 📱 **Responsive** — mobile-first design across all pages
- 🎨 **Premium Design** — Inter + Playfair Display typography, clean minimal aesthetic

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser (or use Live Server)
3. No build step required — it's pure HTML/CSS/JS
